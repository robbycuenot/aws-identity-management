# Example Output

This folder contains a complete example of the Terraform configurations generated by the [aws-identity-management-generator](https://github.com/robbycuenot/aws-identity-management-generator).

All data in this example is fictional and uses placeholder values. It demonstrates the structure and patterns used in a real deployment.

## Structure

```
example/
├── main.tf                      # Root module orchestrating all components
├── providers.tf                 # Provider configuration (local backend)
├── identity_store/              # Users, groups, and memberships
├── managed_policies/            # AWS managed policy references
├── permission_sets/             # Permission sets with policies
└── account_assignments/         # Account-level access assignments
```

## Usage

This example is for reference only. To generate configurations for your own AWS environment:

1. Clone the [generator repository](https://github.com/robbycuenot/aws-identity-management-generator)
2. Configure AWS credentials with read access to IAM Identity Center
3. Run the generator to create configurations based on your current state
